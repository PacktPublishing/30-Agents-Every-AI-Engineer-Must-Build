### Chapter 5 — Foundational Cognitive Architectures (page 118)

**Type:** Code path typo

**What the book says:**

```
git clone https://github.com/PacktPublishing/30-Agents-Every-AI-Engineer-Must-Build.git
cd 30-Agents-Every-AI-Engineer-Must-Build/"Chapter 05"
```

**What it should say:**

```
git clone https://github.com/PacktPublishing/30-Agents-Every-AI-Engineer-Must-Build.git
cd 30-Agents-Every-AI-Engineer-Must-Build/chapter05
```

**Explanation:** The repository uses lowercase folder names without spaces (e.g., `chapter05`, `chapter10`). The book text references `"Chapter 05"` with an uppercase C, a space, and zero-padded number in quotes, which does not match the actual folder name and will produce a "No such file or directory" error when run.

**Discovered:** 2026-04-08
